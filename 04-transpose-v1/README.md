# Transpose v1

Это усложненная версия задачи [Transpose v0](../03-transpose-v0/README.md).
Формулировка остается той же самой, но в задачу добавлены бенчмарки, которые ваше решение должно проходить.
Используйте полученные на занятиях знания о bank conflicts в shared memory.
