# Dot Product

В этой задаче вам предстоит написать кернел, вычисляющий скалярное произведение двух векторов:
$$
A \cdot B = \sum_{i}^{N}{A_iB_i}, \text{ где } N \leq 10^6
$$

Предполагается использование только shared memory.
В этом случае промежуточные результаты вычисления частичных сумм в отдельных блоках придется записывать в массив `workspace`, после чего делать редукцию по нему отдельным кернелом.
Тестирующий код сам выделит под `workspace` память, размер которой определяете вы сами реализацией функции `EstimateDotProductWorkspaceSizeBytes`.

Гарантируется, что входные указатели выровнены по 256 байт.

**NB:** в данной задаче впервые появляются бенчмарки в проверяющей системе.
Запустить локально их можно, собрав и запустив исполняемый файл `bench_dot_product`.
Для успешного прохождения бенчмарков в проверяющей системе ваше решение должно иметь время работы, не превышающее время работы референсного авторского решения более чем на 10%.
