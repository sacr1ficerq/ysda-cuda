# GPU Puzzles #1

В этой задаче вам предстоит написать несколько GPU-кернелов (функций, выполняемых на GPU) с постепенно возрастающей сложностью.

## Map

Напишите кернел `Map`, добавляющий число 10 к каждой позиции во входном массиве и сохраняющий результат в выходном массиве.
На каждую позицию в исходном массиве запускается ровно один CUDA-поток, на всю задачу запускается один блок потоков.

Пример:

```
[1, 2, 3] -> [11, 12, 13]
```

## Zip

Напишите кернел `Zip`, который складывает поэлементно два массива.
На каждую позицию в исходном массиве запускается ровно один CUDA-поток, на всю задачу запускается один блок потоков.

Пример:

```
[1, 2, 3] + [1, 1, 1] -> [2, 3, 4]
```

## Guard

Напишите кернел `Guard`, добавляющий число 10 к каждой позиции во входном массиве и сохраняющий результат в выходном массиве.
**NB:** в этом пункте, в отличие от `Map`, может быть запущено большее число потоков, чем число элементов массива.
При этом на всю задачу так же запускается один блок потоков.

Пример:

```
[1, 2, 3] -> [11, 12, 13]
```

## Block

Напишите кернел `Block`, добавляющий число `value` (передается аргументом) к каждой позиции во входном массиве и сохраняющий результат в выходном массиве.
**NB:** как и в `Guard`, здесь может быть запущено большее число потоков, чем число элементов массива.
Помимо этого, может быть запущено несколько блоков потоков.

Пример:

```
[1, 2, 3] + 42 -> [43, 44, 45]
```
