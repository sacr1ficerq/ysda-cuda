# Quantization

Квантизация - это процесс уменьшения битности параметров модели, чтобы уменьшить объем памяти,
требуемый для хранения этих параметров, и сократить количество вычислений без значительного ухудшения качества работы модели.

Существует множество различных способов квантизации, балансирующих между степенью сжатия, скорости вычислений и точности. Результатом квантизации является квантизованная модель - набор квантизованных весов и скейлов, необходимых для деквантизации.

В этой задаче вам предстоит реализовать одну из подзадач, необходимых для квантизации модели - квантизацию матрицы весов линейного слоя.

Необходимо реализовать способ квантизации матрицы весов `W` с дополнительной балансировкой весов с помощью массива `S` из типа данных `float` в тип данных `int8_t` по следующей формуле:

$$
W'_{r'c'} = \operatorname{RoundNearest}(\frac{(W_{r'c'} + S{c'}) * 127} {\max (mScale, \max_{r=r'}\|W_{rc'} + S{c'}\|)})
$$

где `W'` - квантизованная матрица `int_8t` , `W` - исходная матрица `float`, `S` - дополнительный массив коэфициентов `float`, `mScale` - минимальный возможный скейл (в задаче он равен `1e-5`).

Пример:
```
           W                    S                        W'                     Scales
      1.0 0.1 -0.08          0.2 0 1      =>            127 11 97                105.83333
     -3.3 -0.3  3                                      -98 -10 127               31.75
```

## Необходимые функции и формат данных
Вы должны реализовать следующие функции со следующими сигнатурами:

```cpp
void Quantization(size_t rows, size_t cols, const float* d_inputMatrix, const float* d_balanceFactors, size_t inputStride, size_t outStride, int8_t* d_out, float* d_outScales);
```

где

`rows, cols` - размер входной и выходной матриц. Можно считать, что матрица представлена в формате **RowMajor**. В примере выше rows = 2, cols = 3

`inputStride, outputStride` - расстояние **(в количестве элементов)** между строками. Для примера выше `inputStride == outputStride == cols == 3`, однако в общем случае из-за выравниваний это число может быть больше, чем `cols`.

`d_outScales` - массив длины `rows`.

`d_balanceFactors` - массив длины `cols`

Внутри функции `Quantization` все **аллокации и копирования входов уже произведены**. Все, что должна делать функция `Quantization` - посчитать необходимые параметры (на усмотрение студента) запуска для вызова `QuantizationKernel` и позвать последний.

Гарантируется, что `cols % 4 == 0`
